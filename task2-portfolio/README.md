# Task 2 - Personal Portfolio Website

## 📌 Project Title
SaiReddy-sm-portfolio

## 📖 Description
This is an advanced personal portfolio website developed using HTML, CSS and basic JavaScript.

The portfolio showcases:
- Professional introduction
- Profile image layout
- Skills section
- Project section
- Contact information
- Dark/Light mode toggle
- Smooth animations

The website is fully responsive and works on mobile, tablet and desktop devices.

---

## 🚀 Features
- Animated hero section
- Professional circular profile image layout
- Skill cards with hover animation
- Project showcase section
- Dark Mode Toggle
- Clean modern UI design
- Responsive layout using Flexbox and Media Queries

---

## 💻 Technologies Used
- HTML5
- CSS3
- Flexbox
- Media Queries
- Basic JavaScript (Dark Mode Toggle)

---

## 📱 Responsive Design
This portfolio adapts to:
- Mobile devices
- Tablets
- Desktop screens
