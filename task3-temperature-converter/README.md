## 📖 Description

This project is a responsive temperature converter web application developed using HTML, CSS, and JavaScript.

The application allows users to input a temperature value and convert it between:

- Celsius (°C)
- Fahrenheit (°F)
- Kelvin (K)

The project includes input validation and dynamic result display.

---

## 🎯 Functional Requirements

The user interface includes:

1️⃣ Input field to enter temperature value  
2️⃣ Dropdown menu to select input unit  
3️⃣ Convert button to trigger conversion  
4️⃣ Display area to show converted temperature  

---

## 🌡 Conversion Formulas Used

Celsius → Fahrenheit  
`(°C × 9/5) + 32`

Fahrenheit → Celsius  
`(°F − 32) × 5/9`

Celsius → Kelvin  
`°C + 273.15`

Kelvin → Celsius  
`K − 273.15`
